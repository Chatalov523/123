# GitHub User Finder - Поиск пользователей GitHub 🔍

**Автор:** Шаталов Андрей
**Дата сдачи** 05.05.2026

## Описание

GitHub User Finder - это удобное десктопное приложение для поиска пользователей GitHub, просмотра их профилей и сохранения избранных разработчиков. Приложение использует официальное GitHub API для получения данных и не требует API ключа для базового функционала.

### Основные возможности:

- ✅ **Поиск пользователей** GitHub по username
- ✅ **Отображение подробной информации**:
  - Имя и username
  - Аватар (загружается автоматически)
  - Локация и компания
  - Биография
  - Количество репозиториев
  - Количество подписчиков и подписок
  - Количество Gist
- ✅ **Избранное**:
  - Добавление пользователей в избранное
  - Сохранение в JSON файл
  - Быстрый переход к профилю из избранного
- ✅ **Валидация ввода**:
  - Поле поиска не может быть пустым
  - Обработка ошибок API
- ✅ **Дополнительные функции**:
  - Открытие профиля в браузере
  - Удаление из избранного
  - Автоматическая загрузка и кэширование аватаров
